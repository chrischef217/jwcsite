# JWC 화장품 웹사이트

블랙 & 화이트 컬러의 고급스러운 화장품 기업 웹사이트입니다.

## 📋 프로젝트 개요

- **회사명**: JWC
- **유형**: 화장품 전문 기업 웹사이트
- **컨셉**: 블랙 & 화이트 프리미엄 디자인
- **특징**: 완전한 관리자 시스템, 페이지별 히어로 이미지, 모션 그래픽 텍스트

## 🎯 주요 기능

### ✅ 프론트엔드 (사용자 페이지)

1. **Home (index.html)**
   - 히어로 이미지 슬라이더 (여러 이미지 자동 롤링)
   - 모션 그래픽 텍스트 오버레이 (8가지 애니메이션)
   - 회사 정보 섹션

2. **About (about.html)**
   - 개별 히어로 이미지
   - 회사 소개 및 비전/미션

3. **Products (products.html)**
   - 개별 히어로 이미지
   - 제품 갤러리 (카테고리 필터)

4. **Contact (contact.html)**
   - 개별 히어로 이미지
   - 연락처 정보 및 문의 폼

### 🔐 관리자 시스템 (admin.html)

1. **로고 관리**
   - 파일 업로드 (클릭 또는 드래그앤드롭)
   - 모든 페이지에 자동 표시
   - 이미지 압축 (500px, 90% 품질)

2. **히어로 슬라이더 관리**
   - 여러 개 이미지 업로드
   - 순서 변경 (↑↓ 버튼)
   - 개별 삭제
   - 슬라이드 간격 설정 (1-30초)

3. **히어로 텍스트 관리**
   - 메인 제목/부제목 설정
   - 텍스트 색상 선택
   - 크기 조절 (슬라이더)
   - 그림자 on/off
   - 8가지 애니메이션 선택

4. **페이지 히어로 이미지 관리**
   - About 페이지 히어로 이미지
   - Products 페이지 히어로 이미지
   - Contact 페이지 히어로 이미지
   - 각 페이지별 표시 on/off

5. **제품 관리**
   - 제품 추가/수정/삭제 (예정)

## 🔑 관리자 계정

```
비밀번호: 1111
```

(아이디 없이 비밀번호만으로 로그인)

## 🚀 사용 방법

### 0. 자동 로고 설정 (장원 로고) ⭐ NEW
1. 브라우저에서 `setup-logo.html` 페이지 접속
2. 자동으로 장원 회사 로고가 데이터베이스에 저장됩니다
3. "홈으로 이동" 버튼을 클릭하여 메인 페이지로 이동
4. 모든 페이지 헤더에 장원 로고가 표시됩니다

### 1. 관리자 로그인
1. 헤더 우측 ⚙ 아이콘 클릭 또는 `admin-login.html` 접속
2. 비밀번호: `1111` 입력
3. 로그인 버튼 클릭

### 2. 로고 업로드 (수동)
1. **로고 관리** 메뉴 클릭
2. 클릭 또는 드래그앤드롭으로 로고 파일 업로드
3. **로고 저장** 버튼 클릭
4. 모든 페이지 헤더에 자동 표시됨

### 3. 히어로 슬라이더 설정 (Home 페이지)
1. **히어로 슬라이더** 메뉴 클릭
2. 슬라이드 간격 설정 (예: 5초)
3. 여러 개의 이미지 선택 (Ctrl+클릭 또는 드래그앤드롭)
4. **이미지 저장** 버튼 클릭
5. ↑↓ 버튼으로 순서 변경 가능
6. 🗑️ 버튼으로 개별 삭제 가능

### 4. 히어로 텍스트 설정
1. **히어로 텍스트** 메뉴 클릭
2. 메인 제목과 부제목 입력
3. 텍스트 색상, 크기, 그림자 설정
4. 애니메이션 효과 선택
5. **텍스트 설정 저장** 버튼 클릭

### 5. 페이지 히어로 이미지 설정
1. **페이지 히어로 이미지** 메뉴 클릭
2. About/Products/Contact 중 원하는 페이지 선택
3. 히어로 이미지 표시 체크박스 활성화
4. 이미지 업로드
5. **저장** 버튼 클릭

## 🎨 디자인 특징

### 컬러 시스템
- **블랙**: 헤더, 푸터, 버튼 (#000)
- **화이트**: 배경, 텍스트 (#fff)
- **그레이**: 카드 배경 (#f5f5f5)

### 타이포그래피
- **한글**: Noto Sans KR
- **영문**: -apple-system, BlinkMacSystemFont, Segoe UI

### 애니메이션 (히어로 텍스트)
1. **Fade In Up**: 아래에서 위로 페이드
2. **Fade In Down**: 위에서 아래로 페이드
3. **Zoom In**: 확대되며 등장
4. **Slide Left**: 왼쪽에서 슬라이드
5. **Slide Right**: 오른쪽에서 슬라이드
6. **Typing**: 타이핑 효과
7. **Glitch**: 글리치 효과 (무한 반복)
8. **Floating**: 플로팅 효과 (무한 반복)

## 💾 데이터 저장

### IndexedDB (브라우저 저장) ⭐ 현재 버전
- **저장 위치**: 브라우저 내장 IndexedDB
- **동기화**: 같은 브라우저 내에서만 데이터 공유
- **스토어**:
  - `images`: 로고, 히어로 슬라이더, 페이지 히어로 이미지
    - type: 이미지 타입 (logo, heroSlider, pageHero)
    - page: 페이지 이름 (about, products, contact)
    - data: Base64 인코딩된 이미지 데이터
    - order_index: 정렬 순서
  - `settings`: 앱 설정값
    - slideInterval: 슬라이드 간격
    - {page}HeroEnabled: 페이지별 히어로 활성화 여부

### 이미지 압축
- **히어로 이미지**: 1920px, 85% 품질
- **로고 이미지**: 500px, 90% 품질
- **Base64 저장**: IndexedDB에 보관 (브라우저 내)

## ⚠️ 주의사항

1. **IndexedDB 저장** ⭐ 현재 버전
   - 모든 이미지는 Base64로 인코딩되어 브라우저 IndexedDB에 저장됨
   - 이미지는 자동 압축됨 (1920px, 85% 품질)
   - 로고는 500px, 90% 품질로 압축
   - 브라우저별로 독립적인 데이터 저장
   - **주의**: 브라우저 캐시/데이터 삭제 시 저장된 이미지도 삭제됨

2. **브라우저 호환성**
   - Chrome, Firefox, Safari, Edge 지원
   - IndexedDB 필수 (현대 브라우저)

3. **데이터 지속성**
   - IndexedDB에 브라우저별로 저장
   - 같은 브라우저에서 계속 유지됨
   - **주의**: 시크릿 모드나 브라우저 데이터 삭제 시 손실
