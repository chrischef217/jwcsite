# 🔐 관리자 페이지 접속 방법

## 📍 관리자 버튼 위치

관리자 버튼은 **모든 페이지 하단(푸터) 우측**에 위치해 있습니다.

### 특징
- ⚙️ 작은 톱니바퀴 아이콘
- 투명도가 낮아 일반 사용자들이 쉽게 발견하기 어려움
- 마우스를 올리면 약간 밝아짐
- 크기: 30px (모바일: 25px)
- 위치: 푸터 우측 하단 모서리

### 접속 방법

#### 방법 1: 푸터 버튼 클릭
1. 페이지 맨 하단으로 스크롤
2. 푸터(검은색 배경) 우측 하단의 작은 톱니바퀴(⚙️) 아이콘 클릭
3. 관리자 로그인 페이지로 이동

#### 방법 2: 직접 URL 입력
- 관리자 로그인: `https://8080-i94gkhqz43zzlj510n0w3-0e616f0a.sandbox.novita.ai/admin-login.html`

### 로그인 정보
```
아이디: 1
비밀번호: 1
```

## 🎯 디자인 의도

이 버튼은 **일반 사용자들이 쉽게 찾지 못하도록** 다음과 같이 디자인되었습니다:

1. **위치**: 페이지 맨 하단(일반 사용자들이 잘 확인하지 않는 위치)
2. **색상**: 투명도가 낮은 회색 (눈에 잘 띄지 않음)
3. **크기**: 작은 크기 (30px)
4. **투명도**: 기본 30% 투명도, 호버 시 60%

관리자만 알고 있는 비밀스러운 접근 방법입니다! 🤫
